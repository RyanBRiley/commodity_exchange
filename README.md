# Commodity Exchange
A tokenized commodity exchange.

To run the dev server, simply

```shell_session
  $ cd exchange_react
  $ npm start
```

To create a built version of the app,
```shell_session
  $ cd exchange_react
  $ npm run build
```

To additionally publish the build onto the gh-pages branch,
```shell_session
  $ npm run deploy
```
### Dependencies
* Node.js
* React
* npm
